Selection of Output Results

In order to minimize the size of the batch output report, AutoPIPE provides options to select which loads and load combinations are to be included.

1. Select Home > Results > Combinations, then select the Non-Code Comb. tab.

Disable these options.

  1. Note that by default all of the combinations are enabled. For this tutorial report, disable the

Print option for the following individual load cases: GE1 and GE2.

  1. When the dialog appears as shown above, press OK.
