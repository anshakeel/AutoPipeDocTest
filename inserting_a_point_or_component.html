Inserting a Point or Component

  • Position the cursor on the desired point by clicking on it, then click on one of the Piping Component commands. To insert an intermediate run point, or multiple run points, click on the Run command.
  • Position the cursor on the insertion point, and then select the desired component from the Insert ribon.
  • Users can graphically select a range to insert across ranges of points with one command (see

‘Selecting a Range’)

  • Place the cursor on the desired point, then use the keyboard equivalent commands to key-in the insertion.
