Service Request Manager

You can use the Service Request Manager to report a software problem, ask for help, request a change, or request a new feature. You do not need to be a Bentley Support Services member to use Service Ticket Manager, however you do need to register as a user.

Service Requests (previously Service Tickets) can be created through the Bentley CONNECT Personal Portal:

    1. If the CONNECTION Client is running in the Windows System Tray click it and choose Personal Portal.
    2. Browse to the bottom of the page and click on the New Service Request tile. The Service Request Manager page will open.
    3. Select the Product you would like to create a Service Request for and fill in the other fields.
    4. To add attachments, click the Browse button and select the desired file.
    5. Click Submit to Support button to submit the service request.
