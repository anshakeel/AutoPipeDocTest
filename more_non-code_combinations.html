More Non-Code Combinations

In this section we will define several additional user-defined non-code combinations to be included in the output results. Each combination is specified in the User Non-Code Combinations dialog.

Note:

The Gr+T1 and Gr+T2 non-code combinations are generated as default combinations GT1 and GT2 as shown on the following pages so are not required to be manually

defined.
  1. Select Result > Setup > Combinations to display the Load Combination dialog, then select the

Non-Code Comb. tab.

  1. Click the New button.
  2. The User Non-Code Combinations dialog displays.

  1. Input the following:
Combination Name GR+T1+E2
Combination Method 1 Sum
Case/Comb (1) GR(1) Factor 1.0
Case/Comb (2) T1(1) Factor 1.0
Case/Comb (3) E2(1) Factor 1.4
  1. When the dialog appears as shown above, press OK.
Note: Leave the Load Combination dialog open with the Non-Code Combinations tab active.
  1. Click the New button.

  1. Input the following:
Combination Name GR+T2+E1
Combination Method 1 Sum
Case/Comb (1) GR(1) Factor 1.0
Case/Comb (2) T2(1) Factor 1.0
Case/Comb (3) E1(1) Factor 1.4
  1. When the dialog appears as shown above, press OK.
  2. Click the New button again in the Load Combinations dialog.

  1. Enter the following:
Combination Name GR+T2+E2
Combination Method 1 Sum
Case/Comb (1) GR(1) Factor 1.0
Case/Comb (2) T2(1) Factor 1.0
Case/Comb (3) E2(1) Factor 1.4
  1. When the dialog appears as shown above, press OK. The Non-Code Combinations are listed at the bottom of the grid as shown:

  1. Now that the user-defined non-code combinations are defined, select the Code Comb. tab.
  2. Confirm that your dialog contains the same set of combinations as shown in the figure below. You may need to click the Reset Defaults Only button.

  1. Press OK to close this dialog.
