Modifying Pipe Properties Across a Range

Now that you have created a selection set, you can modify the pipe properties of every component in the range.

  1. Select Modify > Properties > Pipe Properties Over Range. The Pipe Properties dialog is displayed.
  2. From the Pipe Identifier field, select 8"STD {200STD}. An alert dialog is immediately displayed.
