To Select Line Number L100

  1. Use Select > Segments > Line Number and then select L100 from the dropdown list as shown.

Hint: The line number is saved per segment in the model database.
  1. Press OK and notice how segment A (L100) is highlighted in red. This is useful for updating data pertinent to the line such as pressure/temperature data or pipe material properties.
